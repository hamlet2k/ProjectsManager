# Contexto del proyecto

El repositorio se llama **ProjectsManager**.  
Es una aplicación web construida con **Python (Flask)** que permite a un grupo (familia, equipo pequeño) crear proyectos y darles seguimiento.

## Stack y estructura actual
- Backend: Flask + Flask-Migrate para la base de datos.
- Lenguajes principales: Python (~48%), HTML (~51%), Mako.
- Archivos clave: 
  - app.py (punto de entrada de la app, definición de rutas principales).
  - database.py (conexión e inicialización de la BD).
  - forms.py (formularios con validaciones).
  - models/ (clases que representan las tablas de la BD).
  - migrations/ (scripts de migración de esquema con Alembic/Flask-Migrate).
  - templates/ (plantillas HTML).
  - static/ (CSS, JS, imágenes).
  - requirements.txt (dependencias).
- Ejecución actual:
  1. Crear entorno virtual.
  2. Instalar dependencias: `pip install -r requirements.txt`.
  3. Migraciones: `flask db init && flask db migrate && flask db upgrade`.
  4. Run: `flask run`.

## Funcionalidad esperada
- CRUD de proyectos: crear, listar, editar, eliminar.
- Uso de formularios web.
- Persistencia con BD relacional.
- Interfaz web con plantillas HTML.

## Objetivos de actualización (versión inicial)
1. Revisar y documentar los **endpoints** disponibles (qué rutas existen, qué hace cada una).
2. Revisar y documentar los **modelos de datos** (entidades, atributos y relaciones).
3. Modernizar la estructura: usar **Blueprints** para organizar módulos.
4. Mejorar los formularios con validaciones más robustas.
5. Evaluar implementación de **autenticación de usuarios** (login/logout, roles).
6. Preparar la app para deploy moderno: configuración vía `.env`, posible Dockerfile.
7. Mejorar el diseño de plantillas (Bootstrap u otro framework CSS).

## Guía de trabajo
- Siempre mostrar el código actualizado en bloques completos.
- Explicar brevemente los cambios y por qué.
- Mantener compatibilidad con Flask-Migrate.
- Optimizar para claridad y escalabilidad.

Este contexto se irá actualizando a medida que avancemos en las tareas.
